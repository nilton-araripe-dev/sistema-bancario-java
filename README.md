# 🏦 Sistema Bancário em Java

Aplicação simples que simula um sistema bancário, permitindo consultar saldo, realizar depósitos e saques.

## 🚀 Funcionalidades

* Ver saldo atual
* Realizar depósito
* Realizar saque
* Validação de operações (saldo insuficiente e valores inválidos)

## 🛠️ Tecnologias utilizadas

* Java

## ▶️ Como executar o projeto

1. Compile o arquivo:

```
javac Banco.java
```

2. Execute o programa:

```
java Banco
```

3. Escolha a operação desejada no menu exibido no terminal

## 📌 Objetivo

Este projeto foi desenvolvido para praticar conceitos de lógica de programação, estruturas de repetição, condicionais e organização de código em Java.
